## Upload Demo

- Client：React + TS
- Server：Express

## Start Client

```bash
cd client
npm i
npm run dev
```

## Start Server

```bash
cd server
mkdir public/data/uploads # files will be stored here
npm i
npm run start
```

## Features

- singe && multiple file
- base64 file
- multipart
